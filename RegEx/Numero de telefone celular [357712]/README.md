# Expressão Regular (RegEx) para busca de números de telefone celular

## Expressões:

```
(?=\d{2,3}9\d+)(?<!\d)(\d{11,12})(?!\d)
(?<!\d)(\d{2}\s\d{5}\s\d{4})(?!\d)
(?<!\d)(0\d{2}\s\d{5}\s\d{4})(?!\d)
(?<!\d)(\d{2}\s9\s\d{4}\s\d{4})(?!\d)
(?<!\d)(0\d{2}\s9\s\d{4}\s\d{4})(?!\d)
(?<!\d)(\d{9})(?!\d)
(?<!\d)(\d\s\d{8})(?!\d)
(?<!\d)(\(\d{2}\)\d{9})(?!\d)
(?<!\d)(\(0\d{2}\)\d{9})(?!\d)
(?<!\d)(\(\d{2}\)\d{1}\s\d{4}\s\d{4})(?!\d)
(?<!\d)(\(0\d{2}\)\d{1}\s\d{4}\s\d{4})(?!\d)
(?<!\d)(\(\d{2}\)\s\d{1}\s\d{4}\s\d{4})(?!\d)
(?<!\d)(\(0\d{2}\)\s\d{1}\s\d{4}\s\d{4})(?!\d)
```

## Expressão preparada para execução:

```
(?=\d{2,3}9\d+)(?<!\d)(\d{11,12})(?!\d)|(?<!\d)(\d{2}\s\d{5}\s\d{4})(?!\d)|(?<!\d)(0\d{2}\s\d{5}\s\d{4})(?!\d)|(?<!\d)(\d{2}\s9\s\d{4}\s\d{4})(?!\d)|(?<!\d)(0\d{2}\s9\s\d{4}\s\d{4})(?!\d)|(?<!\d)(\d{9})(?!\d)|(?<!\d)(\d\s\d{8})(?!\d)|(?<!\d)(\(\d{2}\)\d{9})(?!\d)|(?<!\d)(\(0\d{2}\)\d{9})(?!\d)|(?<!\d)(\(\d{2}\)\d{1}\s\d{4}\s\d{4})(?!\d)|(?<!\d)(\(0\d{2}\)\d{1}\s\d{4}\s\d{4})(?!\d)|(?<!\d)(\(\d{2}\)\s\d{1}\s\d{4}\s\d{4})(?!\d)|(?<!\d)(\(0\d{2}\)\s\d{1}\s\d{4}\s\d{4})(?!\d)
```
